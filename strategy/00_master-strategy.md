# マスター戦略（v1 / 2026-06-19）

> 戦略責任者：Claude Code ／ 最優先KPI：**収益化速度**

## 0. 現状（データ）
- note 18本・累計PV **245**・最高27PV・売上ほぼ0。→ 原因は商品力でなく**集客ゼロ**。
- 既存資産：GAS/n8n/Python の業務自動化テンプレ多数（制作コスト済み）、X・Threads・noteアカウント保有。
- ペルソナ：**事務作業に追われるひとり社長・個人事業主（AI初心者）**。

## 1. 採用モデル
「**X×Threads で集客 → 既存デジタル商品＋ツールアフィリで即収益化 → 自動生成Shortsでリーチ増幅 → SaaS化**」のハイブリッド。

## 2. 収益化の優先順位（速い順＝この順で力を入れる）
| 優先 | 収益源 | 立ち上げ速度 | 理由 |
|---|---|---|---|
| **1** | 既存テンプレ/デジタル商品（note・将来BOOTH/Gumroad） | 即 | 制作済み・原価ゼロ。流入を入れるだけ |
| **2** | ツール/SaaSアフィリエイト | 数日 | 教えるツールに紹介リンク。既存記事に後付け可・継続報酬 |
| **3** | 新規プロンプト集/テンプレ（量産） | 1〜2週 | ユーザーの制作力が武器。単価UP |
| **4** | 設定代行スポット（ココナラ） | 即(現金) | 手動・スケール頭打ちのため補助扱い。主力にしない |
| **将来** | 自作AIツール → SaaS（月額） | 3〜6ヶ月 | 検証済み需要をプロダクト化。月100万の柱 |

**アフィリエイト方針**：A8.net / もしもアフィリエイト / インフォトップ等に登録し、
①Google Workspace ②各種AI/ノーコードツール ③レンタルサーバー ④AI系スクール の
高関連・継続報酬案件を、既存記事と新規投稿に自然に組み込む。※登録は無料。

## 3. チャネル方針
| チャネル | 役割 | 自動化 | 開始 |
|---|---|---|---|
| **Threads** | 主力集客（テキスト拡散） | 投稿自動化（無料API） | 即 |
| **X** | 第2集客（購買層が濃い） | 当面 無料スケジューラ。API課金は要承認 | 即 |
| **note** | 商品・決済の母艦 | 販売自動 | 稼働中 |
| **Shorts(TikTok/Reels/YT)** | リーチ増幅フィーダー（直接収益でなく送客） | 自動生成→自動投稿（Cursor構築） | 〜2週目 |
| **LINE公式** | 高単価ステップ販売・リスト化 | ステップ配信自動 | 月10万後 |

> 動画判断：このニッチは「考えて買う」商材のため動画は主収益でなくリーチ装置。
> 同一テキストを自動でShorts化し3媒体へ自動投稿（完全自動）→プロフ送客に限定。

## 4. ロードマップ
| フェーズ | 月収 | 主施策 |
|---|---|---|
| P0 立ち上げ(0-1M) | 〜10万 | X/Threads自動投稿開始・無料特典配布・既存商品/アフィリで初収益・自動投稿システム稼働 |
| P1 拡大(1-4M) | 〜30万 | 自動生成Shorts3媒体・商品/アフィリ拡充・高単価/バンドル・LINE導入 |
| P2 飛躍(4-12M) | 〜100万 | 検証済み需要をGPTs/自作ツール→SaaS(月額)・複数チャネル分散で安定 |

## 5. 週次KPI（P0）
- X+Threads 各 1日3投稿（自動）
- 無料特典 配布数 / プロフクリック数 / note流入数
- 初収益（商品 or アフィリ）までの日数 ← 最重要

## 6. 既知のリスク/制約
- X API自動投稿は有料（要承認）。初期は無料スケジューラ。
- SNS自動化は「投稿の自動化はOK・自動いいね/フォロー等のエンゲージ操作はBANリスク」で線引き。
- 動画AIツールは月額課金（要承認）。
