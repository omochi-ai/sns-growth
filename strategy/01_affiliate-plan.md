# アフィリエイト戦略（v1 / 2026-06-19）

> 目的：既存コンテンツに「後付け」で継続収益を足す。新規制作より速い収益源（収益化優先②）。
> 方針：ペルソナ（事務に追われるひとり社長）が**実際に使う/欲しいツール**だけを、信頼を損なわない範囲で紹介。

## 1. 登録するASP（すべて無料・自律推進可 D-005）
| ASP | 特徴 | 優先 |
|---|---|---|
| **A8.net** | 国内最大・案件数最多。サーバー/SaaS/会計ソフトが揃う | ★必須 |
| **もしもアフィリエイト** | W報酬・Amazon/楽天も通せる・初心者向け | ★必須 |
| **インフォトップ** | 情報商材・講座系が高単価。読者層と合致 | ★推奨 |
| afb / バリューコマース / アクセストレード | 案件の取りこぼし回収 | ○余裕があれば |

※ Amazonアソシエイト/楽天は審査・条件あり。ガジェット紹介時に。

## 2. 紹介する案件カテゴリ（関連度＝売れやすさ順）
1. **会計・請求ソフト**（freee / マネーフォワード クラウド 等）
   → 「事務・経理を楽に」と直球。ひとり社長の必需品。**最優先**。
2. **業務自動化/ノーコードSaaS**（自動化ツール、フォーム、CRM 等）
   → 彼女の発信テーマそのもの。記事末尾に自然に置ける。
3. **レンタルサーバー/ドメイン**（エックスサーバー / ConoHa 等）
   → 高単価帯。「自動化・サイト構築」文脈で。
4. **AIツール/ライティング系**（ASP掲載のAIサービス）
   → 「AIで時短」投稿と相性良。
5. **オンライン講座・情報商材**（インフォトップ/Brain）
   → 「副業・自動化を学ぶ」高単価。乱用は信頼を損なうので厳選。

> 報酬額・条件はASP登録後に各案件ページで要確認（本書では固定値を記載しない）。

## 3. 配置場所（どこに貼るか）
- **既存note無料記事**：本文末に「この作業を楽にするツール」枠を新設→該当ツールのリンク（D-004のCTA統一とセットで適用）。
- **X/Threads投稿**：月数回、ノウハウ投稿の流れで自然に。貼りすぎない（売り込み感はフォロー減）。
- **linkpage**：「おすすめツール」セクションを将来追加。
- **無料特典PDF/note**：関連ツールを1〜2点だけ。

## 4. 法令・信頼（必須・厳守）
- **景表法ステマ規制（2023.10〜）**：アフィリンクには **「広告」「PR」「アフィリエイト広告を含む」** 等を**明示必須**。記事冒頭 or リンク近くに表記。
- 使っていない/低品質な商材は紹介しない（1件の不信が全体の売上を壊す）。
- 誇大表現（「絶対稼げる」等）は禁止。

## 5. 着手ステップ（人の手＝ユーザー、判断＝Code）
1. （ユーザー）A8.net ともしもアフィリエイトに無料登録 ※登録のみ。費用なし
2. （Code）登録後、関連度上位の具体案件を選定し、貼る場所・文面を提案
3. （Code/ユーザー）既存記事の末尾に「おすすめツール＋PR表記」を追加
4. クリック計測は将来 `TASKS.md` バックログ「アフィリンク差し込み＆クリック計測」で自動化

> ⚠️ ASP登録・リンク掲載自体は無料。コストは一切発生しない。

### 配置ルール（再掲・厳守）
- リンク近くに **「PR」「広告を含む」** 明記（景表法ステマ規制）。
- 1記事に詰め込みすぎない（1〜2案件）。使っていない物は紹介しない。
- まずは**会計ソフト2件（freee・マネフォ）の提携申請**から。承認されたら既存の請求書/帳票記事に「おすすめ会計ソフト＋PR」を追記。
